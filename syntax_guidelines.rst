Syntax guidelines
=================

Heading 1 
==========
only for top headings, for the separate manuals. i.e. Ultimate 64

.. code::

    Heading 1
    =========

----

Heading 2. 
-----------
Almost all your first headers will be this

.. code::

    Heading 2
    ---------

----

Heading 3.
~~~~~~~~~~
You should not need more than 3 type of headings.
If you do, try to reformat.

.. code::

    Heading 3
    ~~~~~~~~~

----
